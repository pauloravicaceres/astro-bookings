# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-05-02
### Added
- Simple console logging helper with standardized `info`, `warn`, `error`, and `debug` methods.
- Startup, request, and error logs for server boot and request handling.
- Rocket route logging for create, list, not-found, validation, and delete events.

## [0.2.0] - 2026-04-30
### Added
- Rocket management API with create, read, update, delete endpoints.
- Rocket validation for range and passenger capacity.
- End-to-end Playwright tests covering rocket lifecycle and API validation.
